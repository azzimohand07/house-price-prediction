# House Price Prediction

A machine learning project that predicts California house prices using Linear Regression and SGD Regressor.

# Libraries

* Pandas
* NumPy
* Scikit-Learn
* Matplotlib
* Seaborn

# Workflow

* Data preprocessing
* Feature scaling
* Log transformation
* Model training
* Model evaluation
* Learning curve visualization

# Results

| Model             | R² Score | RMSE   |
| ----------------- | -------- | ------ |
| Linear Regression | 0.668    | 67,344 |
| SGD Regressor     | 0.660    | 68,179 |
